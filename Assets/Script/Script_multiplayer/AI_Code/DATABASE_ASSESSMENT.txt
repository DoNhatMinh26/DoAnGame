✅ PHẦN ĐƯỢC ĐÁP ỨNG (6/6 bảng cơ sở)
═════════════════════════════════════════════════════════════════

1. BẢNG: users
   ├─ Mục đích: Quản lý tài khoản người dùng
   ├─ Tính năng đáp ứng:
   │  ✓ Email & Password authentication
   │  ✓ Tên người dùng (username)
   │  ✓ Tuổi (age) → Dùng để xác định level phù hợp
   │  ✓ Avatar (nhân vật) → Mẫu giáo-Lớp 5 có 5 avatar
   │  ✓ Theo dõi thời gian: createdAt, lastLogin
   │  ✓ Trạng thái tài khoản: isActive
   └─

2. BẢNG: player_data
   ├─ Mục đích: Thống kê chính của người chơi
   ├─ Tính năng đáp ứng:
   │  ✓ Tổng điểm (totalScore) → Xếp hạng theo điểm
   │  ✓ Tổng XP (totalXp) → Nâng cấp
   │  ✓ Level hiện tại (currentLevel)
   │  ✓ Xếp hạng (rank) → Liên kết với leaderboard
   │  ✓ Thống kê game: gamesPlayed, gamesWon, winRate
   └─

3. BẢNG: level_progress
   ├─ Mục đích: Tiến độ từng level (Mẫu giáo - Lớp 5)
   ├─ Tính năng đáp ứng:
   │  ✓ Mỗi player có 5-6 level tùy theo cấp độ
   │  ✓ Lưu điểm cao nhất (maxScore)
   │  ✓ Lưu thời gian tốt nhất (bestTime)
   │  ✓ Số lần chơi (attempts)
   │  ✓ XP kiếm từng level
   │  ✓ Hoàn thành hay chưa (completed)
   └─

4. BẢNG: game_history
   ├─ Mục đích: Lịch sử từng ván chơi
   ├─ Tính năng đáp ứng:
   │  ✓ Single mode: Một người chơi đơn
   │  ✓ Multiplayer mode: Chơi với người khác
   │  ✓ Lưu lựa chọn độ khó (easy/normal/hard)
   │  ✓ Thời gian chơi (timeSpent)
   │  ✓ Thời gian timestamp
   │  ✓ Kết quả (win/lose)
   │  ✓ Đối thủ (opponentId) → Khi chơi multiplayer
   └─

5. BẢNG: leaderboard_score
   ├─ Mục đích: Xếp hạng toàn sever theo điểm
   ├─ Tính năng đáp ứng:
   │  ✓ Top 1, 2, 3... người chơi
   │  ✓ Hiển thị avatar của top player
   │  ✓ Automatically update qua player_data
   └─

6. BẢNG: leaderboard_level
   ├─ Mục đích: Xếp hạng theo level cao nhất
   ├─ Tính năng đáp ứng:
   │  ✓ Ai đạt level cao nhất
   │  ✓ Điểm của top level player
   └─
