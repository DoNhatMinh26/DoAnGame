   ├─ Child Controls: Width ✓, Height ✓
   ├─ Child Force Size: Width ✓, Height ✓
   ├─ Spacing: 15-20
   └─ Padding: 30, 30, 30, 30

✨ BƯỚC 3: CHILD ELEMENTS (Components bên trong Panel)
═══════════════════════════════════════════════════════════════════════════════════════

Mỗi Button, Text, Image trong Layout:

Button (Ví dụ)
├─ Add Component: LayoutElement ✓
│  ├─ Preferred Width: -1 (auto/stretch)
│  ├─ Preferred Height: 80 ✓ (fixed height)
│  └─ Line Spacing: On (hoặc preferred)
│
├─ Image (Button background)
│  ├─ Source Image: "button_blue"
│  └─ Preserve Aspect: ✓
│
└─ Text (Button label)
   ├─ Text: "ĐĂNG NHẬP"
   ├─ Font Size: 48 (auto-scale)
   ├─ Alignment: Center Center
   └─ Fit: Preferred Size (TextMeshPro)

Image (Character, Logo)
├─ Add Component: LayoutElement
│  ├─ Preferred Width: 300
│  ├─ Preferred Height: 400
│  └─ Enable Both ✓
│
├─ Add Component: AspectRatioFitter
│  ├─ Aspect Ratio: 0.75 (3:4)
│  └─ Aspect Mode: Fit In Parent ✓
│
└─ Image Component:
   ├─ Source Image: "character_idle"
   └─ Preserve Aspect: ✓

Input Field
├─ Add Component: LayoutElement
│  └─ Preferred Height: 70
│
├─ Input Field Component:
│  ├─ Content Type: Email Address (or Text)
│  ├─ Placeholder: "user@gmail.com"
│  └─ Background Image: "input_bg"
│
└─ TextMeshPro Input Caret:
   ├─ Font Size: 36
   └─ Text Color: (dark)

📊 LAYOUT GROUP CHEAT SHEET
═══════════════════════════════════════════════════════════════════════════════════════

1️⃣ VerticalLayoutGroup (Stack từ trên xuống)
   ││││
   [Btn1]
   [Btn2]
   [Btn3]
   ││││
   
   Setup:
   ├─ Add Component: VerticalLayoutGroup
   ├─ Spacing: 15
   ├─ Child Controls: Width ✓, Height ✓
   ├─ Child Force Size: Width ✓, Height ✓
   └─ Preferred Size: -1, -1
   
   Dùng cho: Menu buttons, input fields, level list

2️⃣ HorizontalLayoutGroup (Xếp từ trái sang phải)
   ─────────────────
   [Btn1] [Btn2] [Btn3]
   ─────────────────
   
   Setup:
   ├─ Add Component: HorizontalLayoutGroup
   ├─ Spacing: 15
   ├─ Child Controls: Width ✓, Height ✓
   ├─ Child Force Size: Width ✓, Height ✓
   └─ Preferred Size: -1, -1
   
   Dùng cho: Answer buttons (4 side by side), avatar grid row

3️⃣ GridLayoutGroup (Lưới rows x columns)
   ─────────────────
   [1] [2] [3]
   [4] [5] [6]
   ─────────────────
   
   Setup:
   ├─ Add Component: GridLayoutGroup
   ├─ Cell Size: 180 x 180
   ├─ Spacing: 15 x 15
   ├─ Constraint: Flexible Rows (hoặc Fixed Column: 3)
   └─ Child ForceSize: On
   
   Dùng cho: Level grid (3x2), avatar selection (2x3)

🔧 COMMON FIXES (Nếu bị lỗi)
═══════════════════════════════════════════════════════════════════════════════════════

❌ Buttons quá nhỏ / quá to:
   → Check LayoutElement: Preferred Height value
   → Thay đổi từ 80 → 100 hoặc 70
   → Đảm bảo "Preferred Height" bị tick ✓

❌ Text bị kéo giãn / méo:
   → TextMeshPro: Font Size = 48 (auto-scale, không cố định)
   → Fit Preferred Size: ✓
   → Layout Element: Height -1 (auto)

❌ Image bị méo (character, logo):
   → Add AspectRatioFitter
   → Set Aspect Ratio đúng (0.75 cho 3:4 ratio)
   → Aspect Mode: Fit In Parent ✓

❌ Components không xếp hàng:
   → Check Parent: Có VerticalLayoutGroup / HorizontalLayoutGroup không?
   → Layout Element: Child Controls ✓ (tất cả phải ON)
   → Spacer images: Add LayoutElement với Flexible Height

❌ Notch (iPhone) che buttons:
   → Canvas Scaler: Respect Safe Area ✓
   → Hoặc thêm Padding top vào Panel

❌ Button không click được:
   → Check Z-order (Button phải trên Background)
   → Image RaycastTarget: ✓
   → Canvas Render Mode: Overlay ✓
