🎨 UI LAYERED STRUCTURE - TẤT CẢ SCREENS (Template)
═══════════════════════════════════════════════════════════════════════════════

HƯỚNG DẪN CHUNG:
1. Mỗi screen có 1 Canvas riêng + 1 SceneManager.cs
2. Các component được liệt kê theo tầp (parent → children)
3. Initial State: ✓ Active (hiển thị) hay ✗ Inactive (ẩn)
4. Z-Order: Thứ tự hiển thị từ phía sau (0) đến phía trước (10)

✅ QUICK REFERENCE: COMPONENT TYPES
═══════════════════════════════════════════════════════════════════════════════

Image              → Background, Icons, Sprites
TextMeshPro        → Tất cả text trên màn hình
Button             → Click-able với callback
InputField         → Nhập input (email, password, etc)
Dropdown           → Dropdown list (Age, Language, etc)
Slider             → Progress bar, Volume control
Toggle             → Checkbox
Panel              → Container (use Image with color)
CanvasGroup        → Để animate alpha/fade in-out
LayoutGroup        → Auto-arrange children (Horizontal/Vertical)
Animator           → Animations (fade, scale, rotate)
ScrollView         → Scrollable list (Leaderboard)
