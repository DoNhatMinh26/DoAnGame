BƯỚC 1: TẠO SCENE MỚI
═══════════════════════════════════════════════════════════════════════════════════════

1. Mở Unity
2. Ở cửa sổ Project (phía dưới/trái):
   ├─ Chuột phải → New Scene
   ├─ Đặt tên: "GameUI"
   └─ Double-click vào GameUI để mở

   Bây giờ bạn thấy 1 scene trắng trong Hierarchy (phía trái)

═══════════════════════════════════════════════════════════════════════════════════════
BƯỚC 2: TẠO CANVAS (khung chứa UI)
═══════════════════════════════════════════════════════════════════════════════════════

Trong Hierarchy (phía trái):
├─ Chuột phải → UI → Canvas

Kết quả: Xuất hiện 1 GameObject tên "Canvas" trong scene

Bây giờ bạn sẽ thấy:
├─ Hierarchy: có "Canvas"
├─ Scene view: 1 khung trắng (Canvas)
└─ Inspector (phía phải): Hiện Canvas settings

═══════════════════════════════════════════════════════════════════════════════════════
BƯỚC 3: CẤU HÌNH CANVAS (Responsive cho tất cả điện thoại)
═══════════════════════════════════════════════════════════════════════════════════════

Canvas đã được tạo sẽ có component:
├─ Canvas (Component)
└─ Canvas Scaler (Component)

Cần thay đổi Canvas Scaler:

Ở Inspector (phía phải), tìm "Canvas Scaler":
├─ UI Scale Mode: Thay đổi từ "Constant Pixel Size" → "Scale With Screen Size"
│  (Click dropdown, chọn "Scale With Screen Size")
│
├─ Reference Resolution: 
│  ├─ X: 1080
│  ├─ Y: 1920
│
├─ Screen Match Mode: Chọn "Match Width Or Height"
│  (Dropdown chọn cái này)
│
└─ Match: Điều chỉnh slider sang phải (= 1)

✓ Xong! Canvas giờ responsive với tất cả điện thoại

═══════════════════════════════════════════════════════════════════════════════════════
BƯỚC 4: TẠO PANEL ĐẦU TIÊN (Welcome Panel)
═══════════════════════════════════════════════════════════════════════════════════════

1. Ở Hierarchy, chuột phải vào "Canvas":
   ├─ UI → Panel → Image

   Kết quả: Xuất hiện "Image" bên trong Canvas

2. Đổi tên thành "WelcomePanel":
   ├─ CliCk phải "Image" → Rename → Gõ "WelcomePanel"

3. Ở Inspector, tìm "Rect Transform":
   ├─ Chuột trái xem "Anchor Presets" (4 mũi tên icon)
   ├─ Hold ALT + Click vào "Stretch" (góc phải dưới) ← 2x2 áp dụng
   
   Kết quả: WelcomePanel bây giờ phủ toàn màn hình

4. Ở Inspector, tìm "Image":
   ├─ Color: Chọn màu muốn (click vào màu trắng)
   └─ Ví dụ: Xanh dương (Blue)

✓ Xong! Lần đầu tiên có 1 Panel màu xanh phủ toàn màn hình

═══════════════════════════════════════════════════════════════════════════════════════
BƯỚC 5: THÊM TEXT (Chào mừng)
═══════════════════════════════════════════════════════════════════════════════════════

1. Ở Hierarchy, chuột phải vào "WelcomePanel":
   ├─ UI → Text - TextMeshPro

   Kết quả: Xuất hiện "Text (TMP)" con bên trong WelcomePanel

2. Đổi tên thành "WelcomeText":
   ├─ Chuột phải → Rename → "WelcomeText"

3. Ở Inspector, tìm "Text Mesh Pro - Text (UI)":
   ├─ Text: Xóa text mặc định, gõ "Chào mừng bạn"

4. Font Size:
   ├─ Font Size: Thay 36 → 48

5. Vị trí (Rect Transform):
   ├─ Pos X: 0
   ├─ Pos Y: 200
   ├─ Width: 800
   ├─ Height: 100

✓ Xong! Thấy "Chào mừng bạn" ở giữa màn hình

Tất cả Panels (13 cái):
☐ 1. WelcomePanel - Chào mừng + 2 buttons (Đăng nhập / Đăng ký)
☐ 2. LoginPanel - Đăng nhập (email, password)
☐ 3. RegisterPanel - Đăng ký (email, username, password, age, avatar)
☐ 4. MainMenuPanel - Menu chính (player info, buttons)
☐ 5. ModeSelectionPanel - Chọn chế độ (Single / Multiplayer)
☐ 6. LevelSelectionPanel - Chọn cấp độ (6 levels)
☐ 7. DifficultySelectionPanel - Chọn độ khó (Easy / Normal / Hard)
☐ 8. InGamePanel - Chơi game (câu hỏi, 4 đáp án, timer, score)
☐ 9. PauseMenuPanel - Tạm dừng (3 buttons: Tiếp tục / Settings / Menu)
☐ 10. GameResultPanel - Kết quả (Win/Lose, điểm, 3 buttons: Replay / Next / Menu)
☐ 11. LeaderboardPanel - Bảng xếp hạng (list, back button)
☐ 12. ProfilePanel - Hồ sơ cá nhân (avatar, stats, back button)
☐ 13. SettingsPanel - Cài đặt (volume, language, back button)

⚠️ LỖI THƯỜNG GẶP
═══════════════════════════════════════════════════════════════════════════════════════

❌ Button / Text không hiển thị:
   → Kiểm tra Pos Y (có thể nằm ngoài màn hình)
   → Kiểm tra Color (có thể là trong suốt)
   → Kiểm tra Parent (phải bên trong Panel hoặc Canvas)

❌ Layout xấu khi đổi resolution:
   → Vì Canvas Scaler chưa cấu hình đúng
   → Kiểm tra: Canvas Scaler = "Scale With Screen Size" ✓

❌ Input field không click được:
   → Nhấn vào, gõ text → Bình thường

❌ Panel không che phủ toàn màn hình:
   → Rect Transform → Anchor Presets → Stretch
