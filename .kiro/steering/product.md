# Product Overview

**DoAnGame** is a Vietnamese math learning game for primary school students (Grades 1–5). Players answer multiple-choice math questions across 100 levels per grade, with difficulty scaling automatically based on grade and level.

## Core Modes
- **Single-player / Quick Play** — no login required; local progress only
- **Multiplayer** — competitive real-time battles via Unity Relay/Lobby; requires Firebase account

## Key Features
- Math question types: addition, subtraction, multiplication, division, finding unknowns (`find_+-`, `find_x:`), and decimals
- Level selection screen with Candy Crush-style wave layout
- Firebase-backed player profiles, leaderboards, and cross-device sync
- 24-hour session tokens for auto-login
- Character name system with uniqueness validation

## Target Platform
Android (primary). Unity Editor on Windows.

## Language
UI text and many variable/comment names are in **Vietnamese**. This is intentional — preserve Vietnamese strings in UI and comments.
