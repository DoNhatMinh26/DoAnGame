🔧 CÁCH SỬA LỖI ANSWERCHOICES SERIALIZATION

=== VẤN ĐỀ ===
Bạn đã gán Answer_0/1/2/3 vào answerChoices trong Inspector
NHƯNG validation vẫn báo "answerChoices: Chưa gán"
→ Đây là Unity serialization bug

=== NGUYÊN NHÂN CÓ THỂ ===
1. Unity không serialize array của custom type đúng cách
2. Scene không save đúng
3. Prefab override conflict
4. Type mismatch (DragAndDrop vs MultiplayerDragAndDrop)

=== GIẢI PHÁP 1: FORCE ASSIGN TRONG RUNTIME ===

✅ ĐÃ TẠO SCRIPT: ForceAssignAnswerChoices.cs

CÁCH DÙNG:
1. Mở scene Test_FireBase_multi.unity
2. Chọn GameplayPanel trong Hierarchy
3. Inspector → Add Component → ForceAssignAnswerChoices
4. Run game
5. Check Console logs:
   - Nếu thấy "✅ Force assigned answerChoices" → THÀNH CÔNG
   - Nếu thấy "❌ Không tìm thấy..." → Có vấn đề với Answer objects

=== GIẢI PHÁP 2: RECREATE ANSWER OBJECTS ===

Nếu ForceAssignAnswerChoices không work:

1. Xóa Answer_0/1/2/3 hiện tại
2. Tạo lại từ đầu:
   - Create Empty GameObject → Rename "Answer_0"
   - Add Component: Image
   - Add Component: MultiplayerDragAndDrop (KHÔNG PHẢI DragAndDrop!)
   - Add Component: Canvas Group
   - Trong MultiplayerDragAndDrop:
     * Gán myText = TextMeshPro child
     * Auto Find Battle Controller = TRUE
   - Duplicate để tạo Answer_1/2/3

3. Gán lại vào UIMultiplayerBattleController:
   - Chọn GameplayPanel
   - Inspector → UIMultiplayerBattleController
   - Kéo Answer_0/1/2/3 vào answerChoices array
   - Save scene (Ctrl+S)

=== GIẢI PHÁP 3: CHECK TYPE MISMATCH ===

QUAN TRỌNG: Answer objects PHẢI dùng MultiplayerDragAndDrop, KHÔNG PHẢI DragAndDrop!

Check:
1. Chọn Answer_0 trong Hierarchy
2. Inspector → Check component type
3. Nếu thấy "DragAndDrop" → SAI! Phải xóa và add "MultiplayerDragAndDrop"
4. Làm tương tự cho Answer_1/2/3

=== GIẢI PHÁP 4: PREFAB OVERRIDE ===

Nếu GameplayPanel là prefab:
1. Chọn GameplayPanel trong Hierarchy
2. Inspector → Overrides dropdown (top right)
3. Click "Apply All" để save changes vào prefab
4. Hoặc "Revert All" rồi gán lại từ đầu

=== DEBUG ===

Sau khi thử các giải pháp trên:
1. Run game
2. Bấm F3 để validate
3. Check Console logs
4. Bấm F4 để export debug report
5. Đọc file BattleDebugReport.txt để xem chi tiết
