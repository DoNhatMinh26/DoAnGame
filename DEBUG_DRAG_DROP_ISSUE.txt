2. THIẾU EVENTSYSTEM

3. RAYCASTTARGET = FALSE

4. CANVASGROUP BLOCKING

   → Unity không detect drag
   FIX: CanvasGroup → blocksRaycasts = TRUE

5. GLOBAL LOCK = TRUE
   → DragAndDrop.SetGlobalLock(true) đã được gọi
   → Tất cả drag bị khóa
   FIX: Gọi DragAndDrop.SetGlobalLock(false)
